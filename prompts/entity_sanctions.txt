## OFAC 50% Rule
If a person on the SDN list owns 50% or more of an entity (directly or indirectly),
that entity is BLOCKED even if not itself listed. This applies to:
- Direct ownership >= 50%
- Aggregate ownership by multiple SDN-listed persons >= 50%
- Indirect ownership through intermediary entities

## Screening Process
1. Search the Consolidated Screening List for the entity name
2. Use screening_list_lookup with datasets parameter to target specific lists:
   - "eu_fsf" — EU Consolidated Financial Sanctions
   - "uk_hmt" — UK HMT Sanctions
   - "au_dfat" — Australian DFAT Sanctions
   - "un_sc" — UN Security Council
   - "ca_sema" — Canadian SEMA Sanctions (includes JVCFOA / Magnitsky Act listings)
3. Search OpenSanctions for the entity via web_search
4. Search OFAC SDN list specifically
5. Check if any beneficial owners are SDN-listed (50% rule)
6. Search for entity aliases and related entities
7. Search ICIJ Offshore Leaks (offshoreleaks.icij.org) for entity name and UBO names
8. Search Interpol Red Notices (interpol.int) for connected persons
9. Search FCA Warning List (fca.org.uk) for unauthorised activities
10. Search Global Affairs Canada sanctions (international.gc.ca) for Canadian autonomous sanctions
11. Search Public Safety Canada listed terrorist entities (publicsafety.gc.ca) per Criminal Code s.83.05
